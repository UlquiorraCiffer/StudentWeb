# Create the Steep design system using the Steep DESIGN.md

## Steep Design System

This repository implements the Steep design system from the DESIGN.md file, which features:
- Light theme with peach accent and warm-gray palette
- Display serif (Signifier) for headlines, sans-serif (Sohne) for UI
- Editorial feel with generous design tokens
- Markdown-based component design

## Key Features

- **Color**: Ink (#17191c), Pure White (#ffffff), Fog (#f7f7f8), Rust accent (#5d2a1a)
- **Typography**: Signifier serif for headlines, Sohne sans-serif for UI
- **Radius**: 24px cards, 9999px buttons
- **Spacing**: 4px base unit, comfortable density

## Project Structure

- `src/components/` - Astro components using Steep design tokens
- `src/layouts/` - Page layouts with Steep styling
- `src/pages/` - Astro pages implementing Steep patterns

## Setup Commands

```bash
npm run dev:steep  # Start dev server with Steep design
npm run build:steep # Build for production with Steep design
```

The Steep site provides a daylight analytics workspace experience with editorial typography and warm accents.
