# Create the Sauce Labs design system using the Sauce Labs DESIGN.md

## Sauce Labs Design System

This repository implements the Sauce Labs design system from the DESIGN.md file, which features:
- Dark theme with neon green pulse accent
- Geometric sans-serif typeface (Aeonik + AeonikFono)
- Developer tooling aesthetic with neon-lit command center
- Isometric illustrations and dark canvas foundation

## Key Features

- **Color**: Obsidian Shell (#132322), Pure White (#ffffff), Mint Frost (#edf7f5), Neon Pulse accent (#3ddc91)
- **Typography**: Aeonik sans-serif for body, AeonikFono for display
- **Radius**: 56px pill buttons, 60px cards
- **Spacing**: 8px base unit, comfortable density

## Project Structure

- `src/components/` - Astro components using Sauce Labs design tokens
- `src/layouts/` - Page layouts with Sauce Labs styling
- `src/pages/` - Astro pages implementing Sauce Labs patterns

## Setup Commands

```bash
npm run dev:labs  # Start dev server with Sauce Labs design
npm run build:labs # Build for production with Sauce Labs design
```

The Sauce Labs site provides a neon-lit engineering console experience with geometric typography and electric accents.
